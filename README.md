An programming language detector

## Install

```bash
npm i program-language-detector
```

## Detectable languages

- JavaScript
- C
- C++
- Python
- Java
- HTML
- CSS
- Ruby
- Go
- PHP

## Usage

```js
const programLanguageDetector = require('program-language-detector')

programLanguageDetector(/* string */)
```
